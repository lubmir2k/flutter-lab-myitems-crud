# flutter-lab-myitems-crud

Flutter learning project - Fresh Flutter template on `main` branch.

## Branches

- **main** - Fresh Flutter counter app template
- **feature/hive-crud** - Hive CRUD implementation with local database

## Get Started

```bash
flutter pub get
flutter run
```

## Switch to CRUD Implementation

```bash
git checkout feature/hive-crud
flutter pub get
dart run build_runner build
flutter run
```
